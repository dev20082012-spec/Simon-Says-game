Simon Says Game 🎮
A browser-based Simon Says game built from scratch using vanilla JavaScript, HTML5, and CSS3 — no frameworks, no libraries.
How to Play

Press any key to start
Watch the sequence of flashing buttons
Repeat the sequence by clicking the buttons in the same order
Each level adds one more step to the sequence
Wrong input? Game over — your score is displayed instantly

What's Under the Hood

Full game sequence logic using arrays (gameSeq, userSeq)
Real-time player input validation on every click
Input-lock flag (started) prevents interaction before game begins
Visual feedback via CSS class toggling for both game flash and user flash
Automatic level progression with 1-second delay after correct sequence
Full state reset on game over — ready to restart immediately

Built With

HTML5
CSS3 (Flexbox layout)
Vanilla JavaScript (DOM manipulation, event listeners, setTimeout)

Run Locally
Just clone the repo and open POJT.html in any browser — no setup needed.
